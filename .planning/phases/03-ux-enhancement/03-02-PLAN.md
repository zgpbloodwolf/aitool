       - `Ctrl+N`（非 Shift）: `e.preventDefault()` -> `chatPanelRef.value?.addNewTab()`
       - `Ctrl+W`: `e.preventDefault()` -> `chatPanelRef.value?.closeTab(activeTabId)`。注意：Plan 01 的 closeTab 已包含活跃检查和确认逻辑。需要获取当前活跃 tabId。方案：在 ChatPanel 的 defineExpose 中额外暴露 `activeTabId` ref 或一个 `getActiveTabId(): string | null` 方法

    **修改 `src/renderer/src/components/ChatPanel.vue`**

    使用原生 HTML5 Drag and Drop API 实现标签拖拽排序（零外部依赖方案）：

    1. 添加拖拽状态变量：
       - `draggedTabId = ref<string | null>(null)`
       - `dragOverTabId = ref<string | null>(null)`

    2. 在模板的 `.tab-item` 上添加拖拽事件：
       - `draggable="true"`
       - `@dragstart="onDragStart($event, tab.id)"`
       - `@dragover.prevent="onDragOver($event, tab.id)"`
       - `@dragenter.prevent="onDragEnter(tab.id)"`
       - `@dragleave="onDragLeave(tab.id)"`
       - `@drop="onDrop($event, tab.id)"`
       - `@dragend="onDragEnd"`

    3. 实现拖拽方法：
       - `onDragStart(e: DragEvent, tabId: string)`: 设置 `draggedTabId.value = tabId`，`e.dataTransfer.effectAllowed = 'move'`
       - `onDragOver(e: DragEvent)`: `e.dataTransfer.dropEffect = 'move'`
       - `onDragEnter(tabId: string)`: 设置 `dragOverTabId.value = tabId`
       - `onDragLeave(tabId: string)`: 如果 `dragOverTabId.value === tabId`，清除
       - `onDrop(e: DragEvent, targetTabId: string)`: 如果 `draggedTabId` 存在且不等于 target，在 `tabs.value` 数组中执行 reordering：从原位置移除 dragged tab，插入到 target 位置
       - `onDragEnd()`: 清除 `draggedTabId` 和 `dragOverTabId`

    4. 添加拖拽相关 CSS：
       - `.tab-item.dragging` 样式：降低 opacity（0.5）
       - `.tab-item.drag-over` 样式：添加左侧或右侧边框指示器（2px solid var(--accent)）
       - 通过 `:class` 动态绑定：`:class="{ dragging: draggedTabId === tab.id, 'drag-over': dragOverTabId === tab.id }`
  </action>
