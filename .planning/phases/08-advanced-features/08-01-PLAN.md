## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Renderer → Main (IPC) | window:register-channel 来自渲染进程，channelId 需校验格式 |
| Main → Renderer (IPC) | broadcastToAll 发送到所有窗口，需确保消息不含敏感数据泄漏 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | S (Spoofing) | window:register-channel IPC | mitigate | 校验 channelId 格式为合法 UUID/自定义格式，拒绝注入特殊字符 |
| T-08-02 | I (Info Disclosure) | broadcastToAll | accept | 广播消息为 Claude 对话内容，非 PII，且所有窗口属于同一用户 |
| T-08-03 | D (DoS) | createWindow 无限制调用 | mitigate | WindowManager 限制最大窗口数（如 10），超出拒绝创建 |
