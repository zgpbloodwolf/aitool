    - 最后一个窗口关闭：根据关闭行为（minimize/quit）处理。如果是 minimize 则隐藏（不销毁），如果是 quit 则允许 app.quit()

    ```typescript
    handleWindowClose(windowId: number): void {
      const win = this.windows.get(windowId)
      if (!win) return

      // 清理频道映射
      this.registry.unregisterWindow(windowId)

      // 如果是最后一个可见窗口
      const visibleWindows = [...this.windows.values()].filter(w => w.isVisible() && w.id !== windowId)
      if (visibleWindows.length === 0) {
        // 最后一个窗口 — 由 tray-manager 的 close 事件处理关闭行为
        // 此处不做额外处理，让 tray 的 mainWindow.on('close') 逻辑决定
      }

      // 从 map 中移除
      this.windows.delete(windowId)
    }
    ```
