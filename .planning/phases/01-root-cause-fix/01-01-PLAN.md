<objective>
Remove the modelUsage and total_cost_usd injection from result messages in claude-webview.ts.

Purpose: The injection (lines 201-214) sets total_cost_usd=0 and modelUsage on every result message. This activates a dormant code path in the Claude Code webview's result handler that checks `total_cost_usd !== undefined`. Before this injection, the field did not exist in Zhipu AI proxy responses, so the webview's usage tracking code path was never entered. Activating it, combined with Zhipu AI's strict concurrency limits, causes 429 errors.

Output: A single modified file with the injection block removed, and the badge in webview-server.ts confirmed unaffected.
</objective>

window.addEventListener('message', function(e) {
  var data = e.data;
  if (!data || data.type !== 'from-extension') return;
  var msg = data.message && data.message.message ? data.message.message : null;
  if (!msg || msg.type !== 'stream_event' || !msg.event) return;
  var usage = msg.event.usage;
  if (!usage) return;
  var total = (usage.input_tokens || 0) + (usage.output_tokens || 0);
  if (total > 0) usedTokens = total;
  updateBadge();
});
